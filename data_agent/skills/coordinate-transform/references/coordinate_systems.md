# 中国常用坐标系指南

## 地理坐标系（经纬度）

| 坐标系 | EPSG | 说明 | 适用场景 |
|--------|------|------|----------|
| CGCS2000 | 4490 | 中国国家大地坐标系 2000 | 国土调查、官方数据 |
| WGS84 | 4326 | GPS 全球坐标系 | GPS 数据、Web 地图、国际交换 |
| GCJ-02 | — | 高德/腾讯加密坐标（火星坐标） | 高德/腾讯地图 API |
| BD-09 | — | 百度加密坐标 | 百度地图 API |

### CGCS2000 vs WGS84
- 差异在厘米级别（<2cm），实际应用中可互换
- 国土调查数据标准为 CGCS2000，但 WGS84 数据可直接使用
- 严格场景下需注明坐标系来源

### GCJ-02 偏移问题
- 相对 WGS84 偏移约 300-500m（非线性偏移）
- 高德/腾讯地图返回的坐标均为 GCJ-02
- **绝不能**将 GCJ-02 坐标当作 WGS84 使用
- 转换需要专门的纠偏算法（非简单平移）

### BD-09 偏移问题
- 在 GCJ-02 基础上再次加密
- 百度地图 API 返回的坐标为 BD-09
- 需先转 GCJ-02 再转 WGS84（两步转换）

## 投影坐标系（平面米制）

### CGCS2000 高斯-克吕格投影
- **3° 带**: 适用于省级及以下精确量测（推荐）
  - 中央经线 = 带号 × 3°
  - 例: 39 带 → 中央经线 117°E → EPSG:4547
  - 中国范围: 25-45 带（75°E ~ 135°E）
- **6° 带**: 适用于全国性概览
  - 中央经线 = 带号 × 6° - 3°
  - 例: 20 带 → 中央经线 117°E → EPSG:4502

### 如何确定高斯带号
```
3° 带号 = round(经度 / 3)
6° 带号 = floor((经度 + 6) / 6)
```

### UTM 投影
- 国际通用 6° 分带
- 中国范围: UTM Zone 43N ~ 53N
- EPSG: 32643 ~ 32653
- 适用于国际合作项目

## 坐标系选择决策表

| 场景 | 推荐坐标系 | 原因 |
|------|-----------|------|
| 数据存储 | EPSG:4490 或 4326 | 通用性好，精度无损 |
| 面积计算 | 当地高斯 3° 带 | 平面坐标，面积准确 |
| 距离计算 | 当地高斯 3° 带或 UTM | 平面坐标，距离准确 |
| Web 地图展示 | EPSG:4326 | Leaflet/Folium 标准 |
| 国土调查数据 | EPSG:4490 | 国家标准 |
| 数据融合/叠加 | 统一为 4326 或 4490 | 避免坐标系混用 |
| 缓冲区分析 | 投影坐标系 | 地理坐标系下缓冲区为椭圆 |

## 常见错误

1. **在地理坐标系下做缓冲区**: 单位是度不是米，1° ≈ 111km，buffer(0.001) ≈ 111m
2. **GCJ-02 当 WGS84 用**: 偏移 300-500m，叠加分析完全错位
3. **高斯带选错**: 数据跨带时需选择中央经线最近的带
4. **面积在地理坐标系下计算**: 结果单位是平方度，无实际意义
5. **忽略 CGCS2000 和 WGS84 的微小差异**: 高精度测量场景需注意
