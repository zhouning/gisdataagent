# 数据审计标准参考

## 第三次全国国土调查字段标准

### 必备字段
| 字段名 | 含义 | 数据类型 | 示例 | 校验规则 |
|--------|------|----------|------|----------|
| DLBM | 地类编码 | VARCHAR(4) | 0101 (水田) | 必须匹配 GB/T 21010-2017 编码表 |
| DLMC | 地类名称 | VARCHAR(50) | 水田 | 必须与 DLBM 编码对应 |
| TBMJ | 图斑面积 | NUMERIC | 1234.56 (㎡) | 与几何面积偏差 ≤5% 或 ≤1m² |
| TBBH | 图斑编号 | VARCHAR(20) | 320000001 | 唯一，不可重复 |
| QSDWDM | 权属单位代码 | VARCHAR(19) | 320116... | 符合行政区划编码规范 |
| ZLDWDM | 坐落单位代码 | VARCHAR(19) | 320116... | 符合行政区划编码规范 |

### 可选但推荐字段
| 字段名 | 含义 | 说明 |
|--------|------|------|
| KCDLBM | 扣除地类编码 | 线状地物扣除 |
| GDLX | 耕地类型 | 梯田/坡耕地等 |
| GDPDJB | 耕地坡度级别 | 1-5 级 |
| TBXHDM | 图斑细化代码 | 二级细化分类 |

## 拓扑检查规则

### 面状要素
| 检查项 | 容差 | 严重程度 | 说明 |
|--------|------|----------|------|
| 重叠检查 | 0.001m² | ERROR | 相邻图斑不应有面积重叠 |
| 间隙检查 | 0.001m | ERROR | 相邻图斑之间不应有间隙 |
| 自相交检查 | — | ERROR | 图斑边界不应自相交 |
| 微面检查 | 0.1m² | WARNING | 极小图斑可能为数据错误 |
| 空几何检查 | — | ERROR | 不应存在空几何记录 |
| 多部件检查 | — | INFO | MultiPolygon 需确认是否合理 |

### 线状要素
| 检查项 | 说明 |
|--------|------|
| 悬挂端点 | 线段端点未与其他线段连接 |
| 伪节点 | 两条线段在非必要处断开 |
| 自相交 | 线段与自身交叉 |

## 面积校验规则

- 计算公式: `deviation = abs(ST_Area(geom_projected) - TBMJ) / TBMJ * 100`
- 通过条件: `deviation ≤ 5%` 或 `abs(ST_Area - TBMJ) ≤ 1m²`（取较宽松者）
- 注意: 面积计算必须在投影坐标系下进行（如 CGCS2000 高斯带）
- 地理坐标系（度）下计算面积会产生严重偏差

## 编码一致性规则

- DLBM 与 DLMC 必须一一对应（如 DLBM="0101" 则 DLMC 必须为 "水田"）
- DLBM 长度必须为 4 位（二级类）或 2 位（一级类）
- 不允许出现非标准编码（如 "011"、"01010" 等）
- QSDWDM 和 ZLDWDM 前 6 位必须为有效行政区划代码
