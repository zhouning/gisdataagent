# 数据审计标准参考

## 第三次全国国土调查字段标准

### 必备字段
| 字段名 | 含义 | 数据类型 | 示例 |
|--------|------|----------|------|
| DLBM | 地类编码 | VARCHAR(4) | 0101 (水田) |
| DLMC | 地类名称 | VARCHAR(50) | 水田 |
| TBMJ | 图斑面积 | NUMERIC | 1234.56 (㎡) |
| TBBH | 图斑编号 | VARCHAR(20) | 320000001 |
| QSDWDM | 权属单位代码 | VARCHAR(19) | 320116... |
| ZLDWDM | 坐落单位代码 | VARCHAR(19) | 320116... |

### 地类编码体系 (GB/T 21010-2017)
- 01: 耕地 (0101 水田, 0102 水浇地, 0103 旱地)
- 02: 园地
- 03: 林地 (0301 乔木林地, 0302 竹林地, 0307 灌木林地)
- 04: 草地
- 05: 商服用地
- 06: 工矿仓储用地
- 07: 住宅用地
- 08: 公共管理与公共服务用地

## 拓扑检查规则

1. **重叠检查**: 相邻图斑不应有面积重叠 (容差 0.001m²)
2. **间隙检查**: 相邻图斑之间不应有间隙 (容差 0.001m)
3. **自相交检查**: 图斑几何不应自相交
4. **微面检查**: 面积 < 0.1m² 的图斑标记为疑似错误

## 面积校验

- 图斑面积 = ST_Area(geometry) 与 TBMJ 属性值对比
- 允许偏差: ≤5% 或 ≤1m²（取较大者）
- 坐标系影响: 面积计算应在投影坐标系下进行
