# 空间统计方法指南

## 方法对照表

| 方法 | 工具函数 | 输入 | 输出 | 适用场景 |
|------|---------|------|------|----------|
| 全局 Moran's I | `spatial_autocorrelation` | 面/点 + 数值字段 | I 值, p 值, Z 值 | 判断整体是否存在空间聚类 |
| 局部 Moran (LISA) | `local_moran` | 面/点 + 数值字段 | 每个要素的聚类类型 | 定位具体聚类区域 |
| Getis-Ord Gi* | `hotspot_analysis` | 面/点 + 数值字段 | 每个要素的 Z/p 值 | 识别热点和冷点 |

## 全局 Moran's I

### 解读规则
| I 值 | p 值 | 结论 |
|------|------|------|
| I > 0 | p < 0.05 | 显著正空间自相关（相似值聚集） |
| I < 0 | p < 0.05 | 显著负空间自相关（相异值聚集） |
| I ≈ 0 | p > 0.05 | 空间随机分布 |

### 注意事项
- I 的期望值 = -1/(n-1)，不是 0
- p < 0.01 为高度显著，0.01-0.05 为显著，>0.05 为不显著
- 样本量 < 30 时结果不可靠

## 局部 Moran (LISA)

### 四象限分类
| 类型 | 含义 | 颜色建议 | 解读 |
|------|------|----------|------|
| HH | 高值被高值包围 | 红色 | 热点聚类区 |
| LL | 低值被低值包围 | 蓝色 | 冷点聚类区 |
| HL | 高值被低值包围 | 粉色 | 空间异常高值 |
| LH | 低值被高值包围 | 浅蓝 | 空间异常低值 |
| NS | 不显著 | 灰色 | 无显著聚类 |

### 显著性过滤
- 默认 p < 0.05 过滤
- 严格分析用 p < 0.01
- Bonferroni 校正: p < 0.05/n（多重比较校正）

## Getis-Ord Gi*

### 热点/冷点判定
| Z 值 | 置信度 | 判定 |
|------|--------|------|
| Z > 2.58 | 99% | 强热点 |
| 1.96 < Z < 2.58 | 95% | 热点 |
| -1.96 < Z < 1.96 | — | 不显著 |
| -2.58 < Z < -1.96 | 95% | 冷点 |
| Z < -2.58 | 99% | 强冷点 |

### 与 LISA 的区别
- Gi* 只识别高值/低值聚集（热点/冷点）
- LISA 还能识别空间异常值（HL/LH）
- 两者结合使用效果最佳

## 空间权重矩阵

### 选择指南
| 类型 | 适用数据 | 说明 |
|------|---------|------|
| Queen | 面状要素 | 共享边或顶点即为邻居（推荐默认） |
| Rook | 面状要素 | 仅共享边为邻居（更严格） |
| KNN | 点/面 | K 个最近邻（K=4-8 常用） |
| 距离阈值 | 点 | 距离内的所有要素为邻居 |

### 常见问题
- **孤岛问题**: 某些要素没有邻居 → 增大距离阈值或用 KNN
- **MAUP**: 分析单元大小影响结果 → 多尺度对比验证
- **边缘效应**: 边界区域邻居不完整 → 结果需谨慎解读
