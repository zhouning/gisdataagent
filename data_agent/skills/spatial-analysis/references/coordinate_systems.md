# 中国常用坐标系指南

## 地理坐标系（经纬度）
- **CGCS2000 (EPSG:4490)**: 中国国家大地坐标系 2000，国土调查数据标准坐标系
- **WGS84 (EPSG:4326)**: GPS 全球坐标系，与 CGCS2000 差异在厘米级别
- **GCJ-02**: 高德/腾讯地图加密坐标系（火星坐标），需专门转换
- **BD-09**: 百度地图加密坐标系，在 GCJ-02 基础上再加密

## 投影坐标系（平面米制）
- **CGCS2000 高斯-克吕格 3° 带**: 适用于省级及以下精确量测
  - 例: EPSG:4547 (中央经线 117°E)
- **UTM 投影**: 国际通用 6° 分带
  - 中国范围: UTM Zone 43N ~ 53N (EPSG:32643 ~ 32653)

## 选择建议
| 场景 | 推荐坐标系 |
|------|-----------|
| 数据融合/叠加分析 | 统一为 EPSG:4326 或 EPSG:4490 |
| 面积/距离计算 | 投影到当地高斯带或 UTM 带 |
| Web 地图展示 | EPSG:4326 (Leaflet/Folium) |
| 国土调查数据 | CGCS2000 (EPSG:4490) |
