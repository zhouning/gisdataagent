# NDVI 与生态评估参考

## NDVI 植被指数

### 计算公式
```
NDVI = (NIR - RED) / (NIR + RED)
```
- NIR: 近红外波段反射率
- RED: 红光波段反射率
- 值域: [-1, 1]

### 解读标准
| NDVI 范围 | 地表类型 | 生态含义 |
|-----------|---------|----------|
| < 0 | 水体、云、雪 | 非植被覆盖 |
| 0 - 0.1 | 裸土、岩石、建筑 | 无植被或极稀疏 |
| 0.1 - 0.2 | 稀疏植被、荒漠 | 植被覆盖度 <15% |
| 0.2 - 0.4 | 草地、灌木 | 植被覆盖度 15-45% |
| 0.4 - 0.6 | 农田、稀疏林地 | 植被覆盖度 45-75% |
| 0.6 - 0.8 | 茂密林地 | 植被覆盖度 75-90% |
| > 0.8 | 热带雨林 | 植被覆盖度 >90% |

### 季节影响
- 春季（3-5月）: NDVI 上升期，适合监测返青
- 夏季（6-8月）: NDVI 峰值期，最佳评估时段
- 秋季（9-11月）: NDVI 下降期，适合监测枯黄
- 冬季（12-2月）: NDVI 最低，常绿/落叶差异明显

## DEM 地形指标

### 坡度分级
| 坡度范围 | 等级 | 土地利用建议 |
|----------|------|-------------|
| 0° - 5° | 平坦 | 适宜耕作、建设 |
| 5° - 15° | 缓坡 | 适宜耕作，需水土保持 |
| 15° - 25° | 中坡 | 限制耕作，宜林/果 |
| 25° - 35° | 陡坡 | 禁止开垦，退耕还林 |
| > 35° | 急坡 | 生态保护区 |

### 坡向解读
| 坡向 | 角度范围 | 特征 |
|------|---------|------|
| 北 (N) | 337.5°-22.5° | 阴坡，温度低，蒸发少 |
| 东北 (NE) | 22.5°-67.5° | 半阴坡 |
| 东 (E) | 67.5°-112.5° | 上午日照 |
| 东南 (SE) | 112.5°-157.5° | 半阳坡，温暖湿润 |
| 南 (S) | 157.5°-202.5° | 阳坡，温度高，蒸发大 |
| 西南 (SW) | 202.5°-247.5° | 半阳坡，午后高温 |
| 西 (W) | 247.5°-292.5° | 下午日照 |
| 西北 (NW) | 292.5°-337.5° | 半阴坡，冬季寒冷 |

## LULC 土地利用/覆盖

### Sentinel-2 10m 分类体系（ESA WorldCover）
| 编码 | 类型 | 说明 |
|------|------|------|
| 10 | 树木覆盖 | 乔木林 |
| 20 | 灌木覆盖 | 灌木丛 |
| 30 | 草地 | 天然草地 |
| 40 | 耕地 | 农田 |
| 50 | 建设用地 | 城镇、道路 |
| 60 | 裸地/稀疏植被 | 荒漠、岩石 |
| 70 | 雪/冰 | 永久积雪 |
| 80 | 水体 | 河流、湖泊 |
| 90 | 湿地 | 沼泽、滩涂 |
| 95 | 红树林 | 热带海岸 |
| 100 | 苔藓/地衣 | 高寒地区 |

### 数据获取
- `download_lulc(boundary_shp, year)`: 下载指定区域和年份的 LULC 数据
- 可用年份: 2017-2024
- 分辨率: 10m（Sentinel-2 源）
- 需要提供边界 Shapefile 作为裁剪范围

## 生态敏感性评价因子

| 因子 | 数据来源 | 分级方法 | 权重建议 |
|------|---------|---------|---------|
| 植被覆盖度 (NDVI) | 遥感影像 | 5 级 (极低-极高) | 0.25 |
| 坡度 | DEM | 5 级 (平坦-急坡) | 0.25 |
| 水域距离 | 水系数据 + 缓冲区 | 5 级 (0-200m 极敏感) | 0.20 |
| 土地利用类型 | LULC | 按生态价值分级 | 0.15 |
| 高程 | DEM | 按海拔分级 | 0.15 |
