# Data Agent 用户手册

## 1. 产品简介
**Data Agent** 是一个基于 AI 代理的地理空间智能分析平台。它专为土地资源规划、生态保护和农业发展设计，能够自动化执行从数据清洗、破碎化评估 (FFI) 到空间布局优化 (DRL) 的全流程。

核心能力：
- **数据诊断**：自动检测 GIS 数据健康度。
- **智能优化**：基于深度强化学习，提供“退耕还林”与“宜耕开发”的空间置换方案。
- **图文报告**：生成专业的三联对比图、交互式地图和 Word 决策报告。

---

## 2. 快速开始

### 2.1 启动界面
访问平台（默认 `http://localhost:8000`）后，您将看到欢迎界面和三个快捷入口：
*   **🗺️ 分析斑竹村数据 (Demo)**：一键运行内置示例数据。
*   **📚 什么是 FFI？**：了解核心指标含义。
*   **🧠 优化理论依据**：查询背后的学术支撑。

### 2.2 上传数据
您也可以点击输入框左侧的 `+` 按钮上传自己的数据。
*   **支持格式**：`.shp` (需包含配套文件), `.zip` (推荐), `.csv`。
*   **CSV 格式要求**：必须包含 `lon/lat` 或 `x/y` 坐标列，以及 `DLMC` (地类名称) 和 `Slope` (坡度) 字段。
*   **SHP 字段要求**：建议包含 `DLMC` 和 `Slope` 字段。

---

## 3. 交互与结果

### 3.1 实时分析流
发送请求后，Agent 会启动分析。您会看到：
*   **Thinking Process (思考过程)**：一个折叠的进度条，点击可展开查看 Agent 调用了哪些工具（如 `calculate_ffi`, `drl_optimize`）。
*   **实时耗时**：每一步操作后都会显示耗时（如 `(1.2s)`）。

### 3.2 可视化结果
分析完成后，对话窗口会自动弹出：
1.  **三联对比图 (Comparison Map)**：
    *   左：优化前现状
    *   中：优化后布局
    *   右：**变化差异图**（🔴红色=退耕还林，🔵蓝色=宜耕开发）
2.  **交互式地图文件**：提供 `.html` 文件下载。下载后用浏览器打开，支持缩放、平移和点击查看地块属性。

### 3.3 导出报告
在对话结束时，屏幕下方会出现 **[📄 导出 Word 报告]** 按钮。
点击后，系统将自动生成一份包含所有分析文字、数据表格和嵌入图片的 `.docx` 文档，供您下载汇报。
