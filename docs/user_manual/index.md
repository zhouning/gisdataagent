# Data Agent 用户手册

## 1. 产品简介
**Data Agent** 是一个基于 AI 代理的地理空间智能分析平台。它专为土地资源规划、生态保护和农业发展设计，能够自动化执行从数据清洗、破碎化评估 (FFI) 到空间布局优化 (DRL) 的全流程。

核心能力：
- **数据诊断**：自动检测 GIS 数据健康度。
- **智能优化**：基于深度强化学习 (Maskable PPO v7)，提供"平衡置换"优化方案。
- **图文报告**：生成专业的三联对比图、交互式地图和 Word 决策报告。
- **自然语言地图控制**：用自然语言显示/隐藏/调整图层样式。
- **协作标注**：在地图上添加和管理团队协作标注。

---

## 2. 快速开始

### 2.1 登录与注册
访问平台（默认 `http://localhost:8000`）后，您将看到登录页面。

- **登录**：输入用户名和密码，默认管理员账户为 `admin` / `admin123`。
- **注册新账户**：点击登录页底部的 "注册新账户" 链接，填写用户名、邮箱（可选）、密码和显示名即可注册。注册成功后自动切换回登录页面。

### 2.2 三面板工作台
登录后进入三面板工作台：

| 面板 | 位置 | 功能 |
|------|------|------|
| **聊天面板** | 左侧 (320px) | 输入分析指令、查看 Agent 执行过程和结果 |
| **地图面板** | 中间 (自适应) | 交互式 Leaflet 地图，支持图层控制、标注、底图切换 |
| **数据面板** | 右侧 (360px) | 5 个标签页：文件、CSV、数据目录、执行历史、用量统计 |

### 2.3 上传数据
点击聊天输入框左侧的 `+` 按钮上传数据：
- **支持格式**：`.shp` (建议打包为 `.zip`)、`.xlsx` / `.csv` (Excel)、`.geojson`、`.gpkg`、`.kml` / `.kmz`
- **Excel/CSV 要求**：包含 "地址" 列或经纬度列（`lon`/`lat`），Agent 自动转换为地图点位
- **文件大小限制**：最大 100 MB

## 3. 交互与结果

### 3.1 实时分析流
发送请求后，Agent 会启动分析。您会看到：
- **思考过程**：折叠的进度条，展开可查看 Agent 调用了哪些工具
- **实时耗时**：每步操作后显示耗时（如 `(1.2s)`）
- **阶段指示**：如 "阶段 2/6: 正在数据质量审计..."

### 3.2 可视化结果
分析完成后，结果自动展示在地图面板和聊天面板中：
1. **三联对比图**（优化管道）：优化前 | 优化后 | 变化差异图
2. **交互式地图**：地图面板自动加载图层，支持缩放、平移、点击查看属性
3. **数据表格**：数据面板 CSV 标签页展示结构化结果

### 3.3 地图操作

#### 图层控制
- 点击地图左上角的图层按钮可显示/隐藏各图层
- **自然语言控制**：直接对 Agent 说 "隐藏人口密度图层" 或 "将耕地图层改为红色"

#### 协作标注
- 点击地图左上角的铅笔按钮进入标注模式
- 在地图上点击任意位置添加标注（填写标题和备注）
- 标注支持团队共享、标记已解决、删除

#### 底图切换
- 点击右上角底图按钮切换底图：CartoDB、OpenStreetMap、高德地图
- 如配置了天地图 Token，还可切换至天地图矢量/影像底图

## 4. 数据面板功能

### 4.1 文件管理
"文件" 标签页显示您上传和生成的所有文件，支持点击查看和管理。

### 4.2 数据目录
"目录" 标签页浏览统一数据湖目录，包含本地、云端和 PostGIS 数据资产。支持按类型筛选、关键词搜索、查看数据血缘。

### 4.3 执行历史
"历史" 标签页查看最近的分析管道执行记录（1天/7天/30天范围选择）。

### 4.4 用量统计
"用量" 标签页查看个人 Token 使用情况：
- **日用量**：今日分析次数 / 每日限额（进度条）
- **月度总量**：本月累计 Token 消耗
- **管道分布**：各管道（优化/治理/通用）的使用占比

## 5. 政务数据治理 (To G)

针对自然资源、规划等政务场景，Agent 提供全生命周期数据治理能力。

### 5.1 核心治理能力
Agent 自动执行以下审计：
- **🛡️ 拓扑体检**：扫描图形重叠、缝隙、自相交等几何错误
- **📋 标准核查**：依据国标检查字段名称、类型及值域
- **🌍 坐标审计**：核查是否为 CGCS2000 坐标系
- **⚖️ 图文对账**：上传 SHP + PDF 报告，自动核对面积指标

### 5.2 治理工作流
1. **审计 (Audit)**：生成审计报告，列出错误清单
2. **修复 (Remediation)**：自动执行重投影、拓扑修复建议
3. **标准化 (Standardization)**：输出符合入库标准的洁净数据

## 6. 空间布局优化 (DRL)

### 6.1 v7 模型特性
- **成对置换**：在减少高坡耕地的同时，在低坡区域开发等量耕地，确保耕地总量基本持平
- **长程规划**：每次优化执行 200 次操作（约 100 对置换）
- **生态优先**：优先移除坡度 >25° 的耕地

### 6.2 使用方法
上传 Shapefile 数据后，发送如下指令：
> "请对这份数据进行空间布局优化分析"

Agent 将自动执行：数据诊断 → 特征工程 → FFI 评估 → DRL 优化 → 可视化 → 报告生成

---

## 7. 导出报告
分析完成后，屏幕下方出现 **[📄 导出 Word 报告]** 按钮。
点击后生成包含分析文字、表格、图表和图片的 `.docx` 文档。

---

## 8. 账户管理

### 8.1 账户设置
点击页面右上角头像，选择 "账户设置" 打开设置面板，可查看：
- 用户名、显示名、角色信息

### 8.2 删除账户
在账户设置的 "危险操作" 区域，可自助删除账户：
1. 点击 "删除我的账户"
2. 输入密码确认
3. 确认删除后，所有数据（文件、分析记录、团队数据）将被永久清除

> **注意**：管理员账户不支持自助删除，需联系其他管理员处理。

---

## 9. 管理后台（仅管理员）

管理员在页面顶部看到 "管理后台" 按钮，点击进入：
- **系统指标**：用户数、分析次数、Token 消耗统计
- **用户管理**：查看所有用户、修改角色（admin/analyst/viewer）、删除用户
- **审计日志**：查看系统操作日志（登录、分析、数据访问等）
